💡 Convertissez votre fichier en CSV ou Excel avec les instructions ci-dessous, puis importez-le ici.

Menu SPSS : Fichier → Exporter → CSV → OK

// Syntaxe SPSS
SAVE TRANSLATE OUTFILE='monfichier.csv'
  /TYPE=CSV /FIELDNAMES /CELLS=VALUES.
EXECUTE.

Menu Stata : File → Save As → CSV

// Commande Stata
use "monfichier.dta", clear
export delimited "monfichier.csv", replace
// Code R — depuis .rds
data <- readRDS("monfichier.rds")
write.csv(data, "monfichier.csv", row.names=FALSE)
// Code R — depuis .sav (package haven)
library(haven)
data <- read_sav("monfichier.sav")
write.csv(data, "monfichier.csv", row.names=FALSE)

Notre Mission

DataSciLab rend l'analyse statistique et scientifique accessible à tous — étudiants, chercheurs et professionnels de santé. Gratuitement, sans compromis sur la qualité.

🔒 Politique de confidentialité

1. Stockage des données

Toutes vos données sont stockées uniquement dans votre navigateur via localStorage. Aucune donnée n'est transmise à des serveurs.

2. Fichiers importés

Vos fichiers CSV/Excel sont traités entièrement dans votre navigateur. Ils ne sont jamais envoyés à des serveurs tiers.

3. Pas de tracking

DataSciLab ne contient aucune publicité et n'utilise aucun outil de tracking.

4. Vos droits

Supprimez toutes vos données depuis Paramètres → "Tout supprimer". Contact : moufsiakou1@gmail.com
